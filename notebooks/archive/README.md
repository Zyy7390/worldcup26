# Notebook Archive

This folder stores sanitized historical notebook snapshots that are available as
project files.

- `worldcup_2026_prediction_bracket.pre_live_update_20260617.ipynb`: pre-live-update notebook snapshot.

Older turn-by-turn notebook states are not reconstructed from chat text alone,
because doing so would risk inaccurate provenance or reintroducing private local
paths from earlier generated artifacts.
